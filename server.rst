The *termy-server* Server
=========================

:program:`termy-server` is a server implementation of the :doc:`TermySequence protocol <protocol>`, written in multithreaded C++ with minimal dependencies.

Use a TermySequence client such as :doc:`qtermy <client-qt>` to interact with the server. Install the server in containers and on remote machines to take advantage of the :term:`connection chaining` feature offered by TermySequence.
